# manager
React-native manager project

# environment setup
- brew install node
- brew install watchman
- sudo npm install -g react-native-cli

# packages
- firebase
- lodash
- react: 16.0.0-alpha.6
- react-native: 0.43.2
- react-router-flux
- react-redux
- redux
- redux-thunk

# run App
- react-native run:ios
